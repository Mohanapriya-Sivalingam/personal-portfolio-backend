# Portfolio Contact Form - Spring Boot Backend

A Spring Boot backend application for handling contact form submissions from a personal portfolio website.
It validates user input, stores data (optional), and sends email notifications using Gmail SMTP.

# 🚀 Features

- ✅ REST API for contact form submission
- ✅ Input validation using Spring Boot Validation
- ✅ Email notification using Gmail SMTP
- ✅ Cross-Origin support for frontend integration (React)

----

# 🛠️ Tech Stack

- Backend: Spring Boot
- Language: Java
- Email: Gmail SMTP (JavaMailSender)
- Build Tool: Maven
- Validation: Hibernate Validator
